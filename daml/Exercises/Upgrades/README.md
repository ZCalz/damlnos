DAML's upgrade mechanism lets you evolve contract templates while preserving existing contracts on the ledger.

### What you will learn

- Adding fields with default values
- Writing an upgrade choice that migrates old contracts to the new template
- Data dependencies — importing a compiled old package into a new one
